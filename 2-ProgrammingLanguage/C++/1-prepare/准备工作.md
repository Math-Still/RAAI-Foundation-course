## C++介绍

C++ 是由比雅尼·斯特劳斯特鲁普（Bjarne Stroustrup）开发的最常用和最受欢迎的编程语言。C++ 是一种高级的面向对象编程语言。这种语言允许开发者编写干净且高效的代码，用于大型应用程序和软件开发、游戏开发以及操作系统编程。C++ 是对 C 编程语言的扩展，加入了面向对象编程（OOPs）的特性，并用于开发计算机程序。这个 C++ 教程将涵盖从基础到高级的所有 C++ 主题，如 C++ 基础、C++ 函数、C++ 类、面向对象编程（OOPs）和 STL 概念。

C++ 是一种非常受欢迎的跨平台编程语言，用于创建高性能的应用程序和软件，如操作系统、游戏、电子商务软件等。它由比雅尼·斯特劳斯特鲁普（Bjarne Stroustrup）开发，是 C 语言的扩展。C++ 提供了对系统资源和内存的高级控制。

### 为什么学习 C++?
- C++ 是最常用和最受欢迎的编程语言之一。
- C++ 被用于开发操作系统、嵌入式系统和图形用户界面。
- 它是一种面向对象的编程语言，实现了所有面向对象编程（OOP）的概念，如抽象、封装和继承，这些特性为程序提供了清- 晰的结构，并允许代码重用，从而降低开发成本并提供安全性。
- 它是可移植的，可以用来创建适应多种平台的应用程序。
- C++ 容易学习，因此你可以选择它作为你的第一个编程语言。
- 由于其语法类似于 C、Java 和 C#，它使得程序员在切换到 C++ 时变得容易。

## 开始安装环境了

我们将在 Linux 上安装 GNU GCC 编译器。所谓的编译器就是处理你写代码的一个工具，让你的代码可以让计算机看的明白。当然不仅只有GNU的编译器，还有LLVM的Clang编译器等。下面我们将介绍如何在Linux系统上安装和使用GCC编译器：

A. 首先，在 Linux 终端中运行以下两个命令：

```bash
sudo apt-get update
sudo apt-get install gcc
sudo apt-get install g++
```
当然，也可以只运行这个命令：
```bash
sudo apt-get install build-essential
```
B. 完成上述步骤后，您应检查 GCC 编译器是否正确安装。为此，请在 Linux 终端中运行以下命令：

```bash
g++ --version
```

对于MacOS系统，下面的指令安装的是Apple相关的开发套件，其中包含了Apple clang编译器。
```bash
xcode-select --install   
```
使用以下命令验证
```bash
clang++ --version
```
当然，对于MacOS用户，也可以安装gcc等其他编译器，但是我还是推荐使用`xcode-select`命令去安装。
