# 1.4 控制流语句

> 原文：[C++ Control Flow](https://en.cppreference.com/w/cpp/language/statements)

> 译者：RAAI学习团队

控制流语句允许程序根据条件执行不同的代码块，或者重复执行某些代码。C++提供了几种控制流语句：条件语句和循环语句。

## if语句

`if`语句是最基本的条件语句：

```cpp
#include <iostream>

int main() {
    int age = 18;
    
    // 简单的if语句
    if (age >= 18) {
        std::cout << "You are an adult." << std::endl;
    }
    
    // if-else语句
    if (age >= 18) {
        std::cout << "You are an adult." << std::endl;
    } else {
        std::cout << "You are a minor." << std::endl;
    }
    
    // if-else if-else语句
    if (age < 13) {
        std::cout << "You are a child." << std::endl;
    } else if (age < 18) {
        std::cout << "You are a teenager." << std::endl;
    } else if (age < 65) {
        std::cout << "You are an adult." << std::endl;
    } else {
        std::cout << "You are a senior." << std::endl;
    }
    
    return 0;
}
```

## switch语句

`switch`语句用于根据表达式的值执行不同的代码块：

```cpp
int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            std::cout << "Monday" << std::endl;
            break;
        case 2:
            std::cout << "Tuesday" << std::endl;
            break;
        case 3:
            std::cout << "Wednesday" << std::endl;
            break;
        case 4:
            std::cout << "Thursday" << std::endl;
            break;
        case 5:
            std::cout << "Friday" << std::endl;
            break;
        case 6:
        case 7:
            std::cout << "Weekend" << std::endl;
            break;
        default:
            std::cout << "Invalid day" << std::endl;
            break;
    }
    
    return 0;
}
```

## while循环

`while`循环在条件为真时重复执行代码块：

```cpp
int main() {
    int count = 0;
    
    // 基本while循环
    while (count < 5) {
        std::cout << "Count: " << count << std::endl;
        count++;
    }
    
    // 无限循环（需要break语句退出）
    count = 0;
    while (true) {
        std::cout << "Infinite loop iteration: " << count << std::endl;
        count++;
        if (count >= 3) {
            break;  // 退出循环
        }
    }
    
    return 0;
}
```

## do-while循环

`do-while`循环至少执行一次代码块，然后检查条件：

```cpp
int main() {
    int number;
    
    do {
        std::cout << "Enter a positive number: ";
        std::cin >> number;
    } while (number <= 0);
    
    std::cout << "You entered: " << number << std::endl;
    
    return 0;
}
```

## for循环

`for`循环是最常用的循环语句：

```cpp
int main() {
    // 基本for循环
    for (int i = 0; i < 5; i++) {
        std::cout << "i = " << i << std::endl;
    }
    
    // 遍历数组
    int numbers[] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++) {
        std::cout << "numbers[" << i << "] = " << numbers[i] << std::endl;
    }
    
    // 基于范围的for循环（C++11）
    for (int num : numbers) {
        std::cout << "num = " << num << std::endl;
    }
    
    return 0;
}
```

## break和continue语句

```cpp
int main() {
    // break语句示例
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break;  // 当i等于5时退出循环
        }
        std::cout << "i = " << i << std::endl;
    }
    
    std::cout << "After break" << std::endl;
    
    // continue语句示例
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue;  // 跳过偶数
        }
        std::cout << "Odd number: " << i << std::endl;
    }
    
    return 0;
}
```

## 嵌套控制流

控制流语句可以嵌套使用：

```cpp
int main() {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            std::cout << i << " is even" << std::endl;
        } else {
            std::cout << i << " is odd" << std::endl;
        }
        
        if (i == 5) {
            std::cout << "Reached 5, continuing..." << std::endl;
            continue;
        }
        
        if (i == 8) {
            std::cout << "Reached 8, breaking..." << std::endl;
            break;
        }
    }
    
    return 0;
}
```

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **无限循环**：
```cpp
int main() {
    int i = 0;
    while (i < 10) {
        std::cout << "i = " << i << std::endl;
        // 忘记递增i，导致无限循环
    }
    return 0;
}
```

2. **缺少break语句**：
```cpp
int main() {
    int day = 1;
    switch (day) {
        case 1:
            std::cout << "Monday" << std::endl;
            // 缺少break，会继续执行下一个case
        case 2:
            std::cout << "Tuesday" << std::endl;
            break;
    }
    return 0;
}
```

3. **逻辑错误**：
```cpp
int main() {
    int x = 5;
    if (x = 10) {  // 使用赋值而不是比较
        std::cout << "x is 10" << std::endl;
    }
    return 0;
}
```
