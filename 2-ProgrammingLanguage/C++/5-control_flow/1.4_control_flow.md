# 1.4 控制流语句

> 原文：[C++ Control Flow](https://en.cppreference.com/w/cpp/language/statements)

> 译者：RAAI学习团队

控制流语句允许程序根据条件执行不同的代码块，或者重复执行某些代码。C++提供了几种控制流语句：条件语句和循环语句。

## if语句

`if`语句是最基本的条件语句。`if`语句（**if statement**）的作用是：判断一个指定的条件是否为真，根据判断的结果决定是否执行另外一条语句。`if` 语句包块两种形式，一种含有**else**分支，另外一种没有。下列是`if语句`的基本使用方式。
```cpp
#include <iostream>

int main() {
    int age = 18;
    
    // 简单的if语句
    if (age >= 18) {
        std::cout << "You are an adult." << std::endl;
    }
    
    // if-else语句
    if (age >= 18) {
        std::cout << "You are an adult." << std::endl;
    } else {
        std::cout << "You are a minor." << std::endl;
    }
    
    // if-else if-else语句
    if (age < 13) {
        std::cout << "You are a child." << std::endl;
    } else if (age < 18) {
        std::cout << "You are a teenager." << std::endl;
    } else if (age < 65) {
        std::cout << "You are an adult." << std::endl;
    } else {
        std::cout << "You are a senior." << std::endl;
    }
    
    return 0;
}
```
简单来说就是
```CPP
if (condition){
    statement
} 
或者
if (condition) {
    statement
} else {
    statement
}
```
在这两个版本的`if`语句中，当**statement**为单个语句时，大括号可以省略。但是**condition**都必须用小括号包围起来。**condition**可以是一个表达式，也可以是一个初始化了的变量声明。无论哪种情况，它们都必须可以转换成`bool`类型。  
在程序比较复杂，例如出现多个if-else嵌套时，可能会出现**悬垂else**的问题。例如：
```cpp
if (grade % 10 >= 3)
    if(grade % 10 > 7)
        lettergrade += '+';
else
    lettergrade += '-';
```
编写这个程序的人本意可能是：给末位是8或者9的成绩添加“+”，末位是3，4，5，6，7的成绩添加“-”。但事实并非如此。这个程序等价于：
```cpp
if (grade % 10 >= 3)
    if(grade % 10 > 7)
        lettergrade += '+';
    else
        lettergrade += '-';
```
显然这和程序员的本意大相径庭。  
多个**if-else**嵌套时，**else**会和距离它最近的尚未匹配其他**else**的**if**配对。假如你理不清它们之间的关系，推荐用大括号把它们分隔开。
```cpp
if (grade % 10 >= 3) {
    if(grade % 10 > 7) {
        lettergrade += '+';
    } else {
        lettergrade += '-';
    }
}
```
这样可能不够优雅，但是更加清晰。
## switch语句

`switch`语句用于根据表达式的值执行不同的代码块。它为我们提供了一条便利的途径使得我们能在若干固定选项中做出选择。下面是一个具体的例子：
```cpp
int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            std::cout << "Monday" << std::endl;
            break;
        case 2:
            std::cout << "Tuesday" << std::endl;
            break;
        case 3:
            std::cout << "Wednesday" << std::endl;
            break;
        case 4:
            std::cout << "Thursday" << std::endl;
            break;
        case 5:
            std::cout << "Friday" << std::endl;
            break;
        case 6:
        case 7:
            std::cout << "Weekend" << std::endl;
            break;
        default:
            std::cout << "Invalid day" << std::endl;
            break;
    }
    
    return 0;
}
```
这个程序实现了判断了 某一天 对应了一周中的星期几。  

`switch`语句首先对括号里的表达式求值，该表达式紧跟在关键字`switch`的后面，可以是一个初始化的变量声明。表达式的值转换成***整数类型***，然后与每个**case**标签的值比较。如果表达式和某个**case**标签的值匹配成功，程序从该标签之后的第一条语句开始执行，直到到达了`switch`的结尾或者是遇到一条**break**语句为止。  

switch有许多**反直觉**的特性：  
1. 括号里的值只能是整数类型，或者是可以转化成整数类型的其他类型。而且它必须是常量表达式。
2. 正常情况下，**break**语句是必要的。我们可以通过例子来说明：
```CPP
    // 假设我们想要写一个统计元音字母格数的程序。
    unsigned int a_cnt = 0;
    unsigned int e_cnt = 0;
    unsigned int i_cnt = 0;
    unsigned int o_cnt = 0;
    unsigned int u_cnt = 0;
    char ch = get_char_val();
    switch (ch) {
        case 'a':
            a_cut++;
        case 'e':
            e_cnt++;
        case 'i':
            i_cnt++;
        case 'o':
            o_cnt++;
        case 'u':
            u_cnt++;
    }
``` 
我们假设输入'e'。此时，程序直接执行**case** 'e'标签后面的代码，`switch`语句此时等价于：
```cpp
    switch (ch) {
        case 'e':
            e_cnt++;
            i_cnt++;
            o_cnt++;
            u_cnt++;
    }
```
简单来说，ch和**case**标签后面的值配对时，程序会执行对应**case**标签后面的语句，而且会同时将语句下面的case标签忽略，但是会保留忽略标签对应的语句。

如果没有任何case标签能匹配上表达式的值，程序将执行紧跟在default标签后面的语句。即使不准备在default标签里执行任何代码，定义一个default标签也是有用的，其目的在于告诉程序的读者，我们已经考虑到了默认的情况，只是在默认情况下我们什么也不做。
## while循环

`while`循环在条件为真时重复执行代码块：

```cpp
int main() {
    int count = 0;
    
    // 基本while循环
    while (count < 5) {
        std::cout << "Count: " << count << std::endl;
        count++;
    }
    
    // 无限循环（需要break语句退出）
    count = 0;
    while (true) {
        std::cout << "Infinite loop iteration: " << count << std::endl;
        count++;
        if (count >= 3) {
            break;  // 退出循环
        }
    }
    
    return 0;
}
```
简单的语法形式如下：
```cpp
    while (condition)
        statement
```
和`if`语句相同，**statement**可以用大括号括起。  

在while结构中，只要condition的求值结果为真就一直执行statement，condition不能为空。如果condition第一次求值就得false，statement一次都不执行。

当不确定到底要迭代多少次时，使用while循环比较合适，比如读取输入的内容。
## do-while循环

`do-while`循环至少执行一次代码块，然后检查条件：

```cpp
int main() {
    int number;
    
    do {
        std::cout << "Enter a positive number: ";
        std::cin >> number;
    } while (number <= 0);
    
    std::cout << "You entered: " << number << std::endl;
    
    return 0;
}
```

## for循环

`for`循环是最常用的循环语句：

```cpp
int main() {
    // 基本for循环
    for (int i = 0; i < 5; i++) {
        std::cout << "i = " << i << std::endl;
    }
    
    // 遍历数组
    int numbers[] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++) {
        std::cout << "numbers[" << i << "] = " << numbers[i] << std::endl;
    }
    
    // 基于范围的for循环（C++11）
    for (int num : numbers) {
        std::cout << "num = " << num << std::endl;
    }
    
    return 0;
}
```
for循环的基本形式如下：
```cpp
    for (init_statement; condition, expression)
        statement
```
关键字`for`及括号里的部分称作for语句头。

**init_statement**必须是以下三种形式中的一种：声明语句、表达式语句或者空语句，因为这些语句都以分号结束。一般情况下，**init_statement**负责初始化一个值，这个值将随着循环的进行而改变。**condition**作为循环控制的条件，只要**condition**为真，就执行一次**statement**。如果**condition**第一次的求值结果就是`false`，则**statement**一次也不会执行。**expression**负责修改**init_statement**初始化的变量，这个变量正好就是**condition**检查的对象，修改发生在每次循环迭代之后。**statement**可以是一条单独的语句也可以是一条复合语句。  

init_statement可以一次定义多个变量，但是声明的变量的类型必须相同。 

`for`语句头的三个表达式都可以省略，省略不同的部分通常有着不同的作用。

C++11引入了一种新的`for`语句, 范围`for`语句。 基本语法如下：
```cpp
    for (declaration : expression)
        statement
```
**expression**必须为一个序列，比如用花括号括起来的初始值列表、数组或者STL中的vector、string等类型的对象。实际上，**expression**的共同特点是拥有能返回迭代器的begin和end成员。  
举几个例子：
```cpp
    int sum = 0;
    for (int n : {1 , 2, 3, 4 , 5, 6}) {
        sum += n;
    }
    std::vector<int> my_vec = {1,2,3,4,5,6}
    for (int n : my_vec) {
        sum += n;
    }
```

## break和continue语句

```cpp
int main() {
    // break语句示例
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break;  // 当i等于5时退出循环
        }
        std::cout << "i = " << i << std::endl;
    }
    
    std::cout << "After break" << std::endl;
    
    // continue语句示例
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue;  // 跳过偶数
        }
        std::cout << "Odd number: " << i << std::endl;
    }
    
    return 0;
}
```
`break`语句负责终止离它最近的while、do while、for循环或switch语句，并从这些语句后面的第一条语句开始执行。

`break`语句只能出现在迭代语句或者switch语句内部（包括嵌套在此类循环里的语句或块的内部）。`break`语句的作用范围仅限于最近的循环或switch语句。

`continue`语句终止最近的循环中的当前迭代并立即开始下一次迭代。`continue`语句只能出现在迭代语句内部（包括嵌套在此类循环里的语句或块的内部）。`continue`语句的作用范围仅限于最近的循环。

`continue`语句中断当前的迭代，但是仍然继续执行循环。对于`for`循环，`continue`语句将执行迭代语句的更新部分。对于`while`和`do while`循环，`continue`语句将跳转到条件判断部分。而对于范围`for`循环，`continue`语句将执行下一次迭代。

## 嵌套控制流

控制流语句可以嵌套使用：

```cpp
int main() {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            std::cout << i << " is even" << std::endl;
        } else {
            std::cout << i << " is odd" << std::endl;
        }
        
        if (i == 5) {
            std::cout << "Reached 5, continuing..." << std::endl;
            continue;
        }
        
        if (i == 8) {
            std::cout << "Reached 8, breaking..." << std::endl;
            break;
        }
    }
    
    return 0;
}
```

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **无限循环**：
```cpp
int main() {
    int i = 0;
    while (i < 10) {
        std::cout << "i = " << i << std::endl;
        // 忘记递增i，导致无限循环
    }
    return 0;
}
```

2. **缺少break语句**：
```cpp
int main() {
    int day = 1;
    switch (day) {
        case 1:
            std::cout << "Monday" << std::endl;
            // 缺少break，会继续执行下一个case
        case 2:
            std::cout << "Tuesday" << std::endl;
            break;
    }
    return 0;
}
```

3. **逻辑错误**：
```cpp
int main() {
    int x = 5;
    if (x = 10) {  // 使用赋值而不是比较
        std::cout << "x is 10" << std::endl;
    }
    return 0;
}
```
