## C++语法

语法指的是在编程语言中编写语句的规则和规定。它们也可以被看作是定义编程语言结构的语法规则。

C++语言也有其自身功能所对应的语法。不同的语句有不同的语法来规定其用法，但 C++ 程序也有一些贯穿所有程序的基本语法规则。

```c++
//头文件
#include <iostream>

// 标准命名空间
using namespace std;

// 主函数
int main() {

    // 输出
    cout << "Hello World!" << endl;

    // 返回语句 C++中可以省略
    return 0;
}
```

使用编辑器写到 `test/helloworld.cpp` 并运行：

```bash
g++ -o ./build/helloworld test/helloworld.cpp
```

./build/helloworld

让我们来解释一下这条编译命令:

1. `g++`是使用g++编译器
2. `-o ./build/helloworld`是将源文件编译成可执行文件helloworld，并且放在./build目录下
3. `test/helloworld.cpp`是输入的在test里面的helloworld.cpp
当然你不理解也没有关系，当前阶段只是会使用这个指令即可。

不同的是编译器可能会使用不同的命令，但是最后应该会产生一个名为`helloworld`的可执行文件，并且你可以运行它。

## 你会看到什么

```bash
./build/helloworld

Hello World!
```

如果没有，则需要返回去修复它。

我们对语法结构进行总结

- 头文件

> 头文件 包含了我们在程序中使用的函数和宏的定义。在 第 1 行，我们使用 #include 语句告诉编译器包含 iostream 头文件库，该库存储了我们用于输入和输出的 cin 和 cout 标准输入/输出流的定义。#include 是一个预处理器指令，我们用它来导入头文件。

- 命名空间

> C++ 中的命名空间用于提供一个作用域或一个区域，在其中我们可以定义标识符。我们使用了 using namespace std 语句，用以指定我们将使用标准命名空间，所有标准库函数都在该命名空间中定义。

- 主函数

> 在 第 3 行，我们将主函数定义为 int main()。函数是任何 C++ 程序中最重要的部分。程序的执行总是从 main 函数开始。所有其他函数都由 main 函数调用。在 C++ 中，main 函数需要返回一个值来表示执行状态。

- 代码块

> 代码块是被 { } 大括号括起来的一组语句。main 函数的主体是被 { } 括起来的部分。

- 分号

> 你可能已经注意到，上面代码中的每条语句后面都有一个 ( ; ) 分号符号。它用于终止程序中的每一行语句。

- 标识符

> 我们使用标识符来命名变量、函数和其他用户定义的数据类型。标识符可以由大写和小写字母、下划线和数字组成。第一个字符必须是下划线或字母。

- 关键字

> 在 C++ 编程语言中，有一些保留字在 C++ 程序中具有特殊的含义。它们不能用作标识符。例如，我们程序中使用的 int、return 和 using 就是一些关键字。

- 基本输出 cout

> 我们使用了 cout 流对象（在头文件中声明）将一个一个字符串打印到标准输出流（stdout）。

- 注释

> 可以在代码中看到带有两个正斜杠 (//) 和一些文本的内容，这在 C++ 程序中被称为注释。注释提供了关于代码或特定代码行的信息，以提高可读性。

## C++ 注释

在 C++ 中，注释用于解释代码并使其更具可读性。在测试替代代码时，注释还可以防止代码的执行。注释的目的是提供有关代码行的信息。程序员通常使用注释来记录他们的工作。

在 C++ 中，注释用于总结算法、标识变量的目的，或澄清看起来不清楚的代码段。注释的用途还包括：

- 注释用于更轻松的调试。
- 它使程序更具可读性，并提供代码的整体描述。
- 注释有助于跳过代码的某些部分的执行。
- 每当一个程序或代码在长时间后被重用时，注释可以快速回顾代码的所有信息。

### 单行注释

在 C++ 中，单行注释用 // 双斜杠表示。它仅对单行代码应用注释。编译器会忽略 // 之后的任何文本，这部分文本不会被执行。

```c++
// C++ 程序演示单行注释
#include <iostream>
using namespace std;
int main() {
    // cout << "note!"; 单行注释，编译器将忽略它
    cout << "note!";
    return 0;
}
```

输出：

```bash
note!
```

### 多行注释

多行注释可以占据多行代码，它以 / ** 开始，以 / 结束，但不能嵌套。任何在 / 和 / 之间的文本都会被编译器忽略。

```c++
/*
    多行注释
    .
    .
    .
*/
// C++ 程序演示多行注释
#include <iostream>
using namespace std;
int main() {
    /* 
    多行注释
    编译器将忽略它 
    cout << "note!";
    */
    cout << "note!";
    return 0;
}
```

输出：

```bash
note!
```

> 作为编译器的一部分，词法分析器扫描字符并将其转换为标记，而不会将注释文本传递给解析器。由于注释不对程序的功能做出贡献，因此在编译时会被简单地忽略。因此，我们可以理解注释只是程序中的文本，编译器会忽略它们。
>
## 标识符

在 C++ 编程语言中，标识符是分配给变量、函数、类、结构体或程序中其他实体的唯一名称。例如，在以下语句中：

```c++
int num = 1;
```

**num** 是一个标识符。

### C++ 中标识符命名规则

我们可以使用任何单词作为标识符，只要它遵循以下规则：

1. 标识符可以由 字母（A-Z 或 a-z）、数字（0-9）和 \*\*下划线 \(\_\) \*\* 组成。特殊字符和空格是不允许的。
2. 标识符只能以 字母或下划线 开头。
3. C++ 有一些 关键字 是保留的，不能用作标识符，因为它们在语言中有预定义的意义。例如，int 不能作为标识符使用，因为它在 C++ 中已有预定义的含义。尝试使用这些关键字作为标识符会导致编译错误。
4. 标识符在其命名空间中必须 唯一。

此外，C++ 是区分大小写的语言，因此标识符如 Num 和 num 被视为不同的标识符。

## 关键字

C++ 是一种强大的语言。在 C++ 中，我们可以编写结构化程序和 面向对象的程序。C++ 是 C 的超集，因此大多数 C 的结构在 C++ 中是合法的，其含义不变。然而，也有一些例外和新增内容。

当编译器处理 C++ 程序的源代码时，每组由空格分隔的字符被称为**标记**。标记是程序中最小的独立单位。C++ 程序是通过使用标记来编写的。它包括以下标记：

- 关键字
- 标识符
- 常量
- 字符串
- 运算符

关键字（也称为 保留字）对 C++ 编译器 有特殊的含义，并且总是以小写字母书写。关键字是语言用于特定目的的词汇，如 void、int、public 等。它不能用作变量名、函数名或任何其他标识符。总共有 95 个保留关键字。以下是一些常用 C++ 关键字的表格。

| C++ 关键字 |        |           |          |
|------------|--------|-----------|----------|
| asm        | double | new       | switch   |
| auto       | else   | operator  | template |
| break      | enum   | private   | this     |
| case       | extern | protected | throw    |
| catch      | float  | public    | try      |
| char       | for    | register  | typedef  |
| class      | friend | return    | union    |
| const      | goto   | short     | unsigned |
| continue   | if     | signed    | virtual  |
| default    | inline | sizeof    | void     |
| delete     | int    | static    | volatile |
| do         | long   | struct    | while    |

以下是对于关键词的解释
> asm: 用于声明一段代码块将传递给汇编器。<br/>auto: 存储类说明符，用于在块中定义对象。<br/>break: 终止 switch 语句或循环。<br/>case: 在 switch 语句中用于指定匹配语句表达式的情况。<br/>catch: 指定异常发生时采取的动作。<br/>char: 基本数据类型，用于定义字符对象。<br/>class: 用于声明一个用户定义的类型，封装数据成员和操作或成员函数。<br/>const: 用于定义在程序执行过程中值不会改变的对象。<br/>continue: 将控制转移到循环的开始。<br/>default: 处理 switch 语句中未被 case 处理的表达式值。<br/>delete: 内存释放操作符。<br/>do: 表示 do-while 语句的开始，其中子语句重复执行，直到表达式的值为逻辑假。<br/>double: 基本数据类型，用于定义浮点数。<br/>else: 在 if-else 语句中使用。<br/>enum: 用于声明用户定义的枚举数据类型。<br/>extern: 作为 extern 指定的标识符具有外部链接到块的特性。<br/>float: 基本数据类型，用于定义浮点数。<br/>for: 表示开始一个语句以实现重复控制。<br/>friend: 一个类或操作，其实现可以访问类的私有数据成员。<br/>goto: 将控制转移到指定标签。<br/>if: 表示 if 语句的开始，以实现选择控制。<br/>inline: 函数说明符，指示编译器优选内联替换函数体，而不是通常的函数调用实现。<br/>int: 基本数据类型，用于定义整数对象。<br/>long: 数据类型修饰符，定义 32 位整数或扩展的 double。<br/>new: 内存分配操作符。<br/>operator: 用于重载 C++ 运算符的新声明。<br/>private: 声明类成员在类外不可见。<br/>protected: 声明类成员对派生类可见，其他类不可见。<br/>public: 声明类成员在类外可见。<br/>register: 存储类说明符，是 auto 说明符的一种，但还指示编译器对象将被频繁使用，因此应保存在寄存器中。<br/>return: 将对象返回给函数的调用者。<br/>short: 数据类型修饰符，定义 16 位整数。<br/>signed: 数据类型修饰符，指示对象的符号存储在高位。<br/>sizeof: 返回对象的字节大小。<br/>static: 定义的静态对象的生命周期贯穿整个程序执行。<br/>struct: 用于声明封装数据和成员函数的新类型。<br/>switch: 用于“switch 语句”。<br/>template: 参数化或泛型类型。<br/>this: 指向类的对象或实例的类指针。<br/>throw: 生成一个异常。<br/>try: 表示异常处理程序块的开始。<br/>typedef: 另一个基本或用户定义类型的同义词。<br/>union: 类似于结构体，能够存储不同类型的数据，但一次只能存储一个成员。<br/>unsigned: 数据类型修饰符，指示高位用于对象。<br/>virtual: 函数说明符，声明一个成员函数，该函数将由派生类重新定义。<br/>void: 没有类型或函数参数列表。<br/>volatile: 定义一个值可能以编译器无法检测的方式变化的对象。<br/>while: while 语句的开始和 do-while 语句的结束。<br/>
