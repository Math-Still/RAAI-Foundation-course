# 1.2 变量和数据类型

> 原文：[C++ Variables and Data Types](https://en.cppreference.com/w/cpp/language/variables)

> 译者：RAAI学习团队

在C++中，变量是存储数据的容器。每个变量都有一个类型，这个类型决定了变量可以存储什么样的数据以及占用多少内存空间。

## 基本数据类型

C++提供了几种基本的数据类型：

```cpp
#include <iostream>

int main() {
    // 整数类型
    int age = 25;                    // 整数
    short small_number = 100;        // 短整数
    long big_number = 1000000L;      // 长整数
    long long very_big = 1000000000LL; // 超长整数
    
    // 浮点类型
    float pi_approx = 3.14f;         // 单精度浮点数
    double pi_precise = 3.14159265359; // 双精度浮点数
    long double pi_very_precise = 3.14159265359L; // 长双精度
    
    // 字符类型
    char grade = 'A';                // 字符
    unsigned char byte_value = 255;  // 无符号字符
    
    // 布尔类型
    bool is_student = true;          // 布尔值
    
    // 输出变量值
    std::cout << "Age: " << age << std::endl;
    std::cout << "Grade: " << grade << std::endl;
    std::cout << "Is student: " << is_student << std::endl;
    
    return 0;
}
```

## 变量声明和初始化

在C++中，有几种方式可以声明和初始化变量：

```cpp
int main() {
    // 传统初始化
    int x = 10;
    
    // 构造函数初始化
    int y(20);
    
    // 统一初始化（C++11）
    int z{30};
    
    // 列表初始化
    int w = {40};
    
    // 自动类型推导（C++11）
    auto auto_var = 50;
    auto auto_string = "Hello";
    
    return 0;
}
```

## 你会看到什么

当你运行上面的程序时，你会看到：

```sh
$ make exercise1_2
g++ -Wall -g exercise1_2.cpp -o exercise1_2
$ ./exercise1_2

```

注意布尔值`true`被输出为`1`，这是因为`std::cout`默认将布尔值转换为整数输出。

## 类型大小和范围

不同的数据类型占用不同的内存空间：

```cpp
#include <iostream>
#include <climits>

int main() {
    std::cout << "数据类型大小：" << std::endl;
    std::cout << "char: " << sizeof(char) << " bytes" << std::endl;
    std::cout << "int: " << sizeof(int) << " bytes" << std::endl;
    std::cout << "float: " << sizeof(float) << " bytes" << std::endl;
    std::cout << "double: " << sizeof(double) << " bytes" << std::endl;
    
    std::cout << "\n整数范围：" << std::endl;
    std::cout << "INT_MIN: " << INT_MIN << std::endl;
    std::cout << "INT_MAX: " << INT_MAX << std::endl;
    
    return 0;
}
```

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **整数溢出**：
```cpp
int main() {
    int max_int = INT_MAX;
    std::cout << "Max int: " << max_int << std::endl;
    max_int = max_int + 1;  // 溢出
    std::cout << "After overflow: " << max_int << std::endl;
    return 0;
}
```

2. **未初始化变量**：
```cpp
int main() {
    int uninitialized;
    std::cout << "Uninitialized: " << uninitialized << std::endl;
    return 0;
}
```

3. **类型转换问题**：
```cpp
int main() {
    int x = 5;
    int y = 2;
    double result = x / y;  // 整数除法
    std::cout << "Integer division: " << result << std::endl;
    
    double result2 = static_cast<double>(x) / y;  // 浮点除法
    std::cout << "Float division: " << result2 << std::endl;
    return 0;
}
```
