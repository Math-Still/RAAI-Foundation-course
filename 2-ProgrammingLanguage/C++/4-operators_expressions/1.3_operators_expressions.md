# 1.3 运算符和表达式

> 原文：[C++ Operators and Expressions](https://en.cppreference.com/w/cpp/language/operators)

> 译者：RAAI学习团队

运算符是C++中用于执行各种操作的符号。表达式是由运算符、操作数和函数调用组成的组合，它们会产生一个值。

## 算术运算符

C++提供了基本的算术运算符：

```cpp
#include <iostream>

int main() {
    int a = 10;
    int b = 3;
    
    // 基本算术运算
    std::cout << "a = " << a << ", b = " << b << std::endl;
    std::cout << "a + b = " << a + b << std::endl;  // 加法
    std::cout << "a - b = " << a - b << std::endl;  // 减法
    std::cout << "a * b = " << a * b << std::endl;  // 乘法
    std::cout << "a / b = " << a / b << std::endl;  // 整数除法
    std::cout << "a % b = " << a % b << std::endl;  // 取余
    
    // 浮点除法
    double c = 10.0;
    double d = 3.0;
    std::cout << "c / d = " << c / d << std::endl;
    
    return 0;
}
```

## 赋值运算符

赋值运算符用于给变量赋值：

```cpp
int main() {
    int x = 5;
    
    // 基本赋值
    x = 10;
    std::cout << "x = " << x << std::endl;
    
    // 复合赋值运算符
    x += 5;   // 等价于 x = x + 5
    std::cout << "x += 5: " << x << std::endl;
    
    x -= 3;   // 等价于 x = x - 3
    std::cout << "x -= 3: " << x << std::endl;
    
    x *= 2;   // 等价于 x = x * 2
    std::cout << "x *= 2: " << x << std::endl;
    
    x /= 4;   // 等价于 x = x / 4
    std::cout << "x /= 4: " << x << std::endl;
    
    x %= 3;   // 等价于 x = x % 3
    std::cout << "x %= 3: " << x << std::endl;
    
    return 0;
}
```

## 比较运算符

比较运算符用于比较两个值：

```cpp
int main() {
    int a = 5;
    int b = 10;
    
    std::cout << "a = " << a << ", b = " << b << std::endl;
    std::cout << "a == b: " << (a == b) << std::endl;  // 等于
    std::cout << "a != b: " << (a != b) << std::endl;  // 不等于
    std::cout << "a < b: " << (a < b) << std::endl;    // 小于
    std::cout << "a > b: " << (a > b) << std::endl;    // 大于
    std::cout << "a <= b: " << (a <= b) << std::endl;  // 小于等于
    std::cout << "a >= b: " << (a >= b) << std::endl;  // 大于等于
    
    return 0;
}
```

## 逻辑运算符

逻辑运算符用于组合布尔表达式：

```cpp
int main() {
    bool x = true;
    bool y = false;
    
    std::cout << "x = " << x << ", y = " << y << std::endl;
    std::cout << "x && y: " << (x && y) << std::endl;  // 逻辑与
    std::cout << "x || y: " << (x || y) << std::endl;  // 逻辑或
    std::cout << "!x: " << (!x) << std::endl;          // 逻辑非
    std::cout << "!y: " << (!y) << std::endl;          // 逻辑非
    
    return 0;
}
```

## 递增和递减运算符

```cpp
int main() {
    int a = 5;
    
    // 前缀递增
    std::cout << "a = " << a << std::endl;
    std::cout << "++a = " << ++a << std::endl;  // 先递增，再使用
    std::cout << "a = " << a << std::endl;
    
    // 后缀递增
    a = 5;
    std::cout << "a = " << a << std::endl;
    std::cout << "a++ = " << a++ << std::endl;  // 先使用，再递增
    std::cout << "a = " << a << std::endl;
    
    // 递减运算符类似
    a = 5;
    std::cout << "--a = " << --a << std::endl;
    std::cout << "a-- = " << a-- << std::endl;
    
    return 0;
}
```

## 运算符优先级

运算符有不同的优先级，这决定了表达式的求值顺序：

```cpp
int main() {
    int result = 2 + 3 * 4;  // 先乘后加
    std::cout << "2 + 3 * 4 = " << result << std::endl;
    
    result = (2 + 3) * 4;    // 括号改变优先级
    std::cout << "(2 + 3) * 4 = " << result << std::endl;
    
    return 0;
}
```

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **除零错误**：
```cpp
int main() {
    int x = 10;
    int y = 0;
    int result = x / y;  // 运行时错误
    std::cout << result << std::endl;
    return 0;
}
```

2. **整数溢出**：
```cpp
int main() {
    int max_int = INT_MAX;
    int overflow = max_int + 1;
    std::cout << "Overflow: " << overflow << std::endl;
    return 0;
}
```

3. **浮点数精度问题**：
```cpp
int main() {
    double a = 0.1;
    double b = 0.2;
    double c = a + b;
    std::cout << "0.1 + 0.2 = " << c << std::endl;
    std::cout << "c == 0.3: " << (c == 0.3) << std::endl;
    return 0;
}
```
