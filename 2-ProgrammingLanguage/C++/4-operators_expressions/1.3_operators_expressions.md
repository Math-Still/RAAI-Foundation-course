# 1.3 运算符和表达式

> 原文：[C++ Operators and Expressions](https://en.cppreference.com/w/cpp/language/operators)

> 译者：RAAI学习团队

在 C++ 里，运算符就像一个个小巧的 “工具”，帮你对数据做加减、比较等操作；而表达式则是这些工具和数据的组合，像搭积木一样凑出一个结果。

在 C++ 中，运算符有以下几种类型：

1. **算术运算符**
2. **关系运算符**
3. **逻辑运算符**
4. **位运算符**
5. **赋值运算符**
6. **自增和自减运算符**
7. **三元运算符**

## 算术运算符

| 运算符       | 功能 |
| :----------- | :--- |
| `+` （单目） | 正   |
| `-` （单目） | 负   |
| `*` （双目） | 乘法 |
| `/`          | 除法 |
| `%`          | 取模 |
| `+` （双目） | 加法 |
| `-` （双目） | 减法 |

算术运算符中有两个单目运算符（正、负）以及五个双目运算符（乘法、除法、取模、加法、减法），其中单目运算符的优先级最高。

C++提供了基本的算术运算符：

```cpp
#include <iostream>

int main() {
    int a = 10;
    int b = 3;
    
    // 基本算术运算
    std::cout << "a = " << a << ", b = " << b << std::endl;
    std::cout << "a + b = " << a + b << std::endl;  // 加法
    std::cout << "a - b = " << a - b << std::endl;  // 减法
    std::cout << "a * b = " << a * b << std::endl;  // 乘法
    std::cout << "a / b = " << a / b << std::endl;  // 整数除法
    std::cout << "a % b = " << a % b << std::endl;  // 取余
    
    // 浮点除法
    double c = 10.0;
    double d = 3.0;
    std::cout << "c / d = " << c / d << std::endl;
    
    return 0;
}
```

> [!NOTE]
>
> 对于双目算术运算符，当参与运算的两个变量类型相同时，不发生类型转换，运算结果将会用参与运算的变量的类型容纳，否则会发生类型转换，以使两个变量的类型一致。
>
> 例如，对于一个整型（`int`）变量 `x`和另一个双精度浮点型（`double`）类型变量 `y`：
>
> - `x/3` 的结果将会是整型；
> - `x/3.0` 的结果将会是双精度浮点型；
> - `x/y` 的结果将会是双精度浮点型；
> - `x*1/3` 的结果将会是整型；
> - `x*1.0/3` 的结果将会是双精度浮点型；

## 关系运算符

在 C++ 中，**关系运算符**用于比较两个值之间的关系，其运算结果是一个布尔值（`true` 或 `false`）。它们是构建条件判断和逻辑控制的基础工具，常见的关系运算符有以下 6 种：

| **运算符** | **名称** | **功能描述**               | **示例** | **结果** |
| ---------- | -------- | -------------------------- | -------- | -------- |
| `==`       | 等于     | 判断两值是否相等           | `5 == 3` | `false`  |
| `!=`       | 不等于   | 判断两值是否不等           | `5 != 3` | `true`   |
| `>`        | 大于     | 判断左值是否大于右值       | `5 > 3`  | `true`   |
| `<`        | 小于     | 判断左值是否小于右值       | `5 < 3`  | `false`  |
| `>=`       | 大于等于 | 判断左值是否大于或等于右值 | `5 >= 3` | `true`   |
| `<=`       | 小于等于 | 判断左值是否小于或等于右值 | `5 <= 3` | `false`  |

## 逻辑运算符

| **运算符** | **功能** | **示例**         | **运算过程**                              | **结果**       |    
| ---------- | -------- | ---------------- | ----------------------------------------- | -------------- |
| `&&`       | 逻辑与   | `true && false`  | 两个操作数都为 `true` 时结果为 `true`     | `false`    |  
| `||`      | 逻辑或   | `true || false`  | 两个操作数有一个为 `true` 时结果为 `true` | `true`|
| `!`       | 逻辑非   | `!true` `!false` | 取反操作：`true` 变 `false`，反之亦然     | `false` `true` |  

## 位运算符

| **运算符** | **功能**       | **示例**                                    | **运算过程**                           | **二进制结果**       | **十进制结果** |
| ---------- | -------------- | ------------------------------------------- | -------------------------------------- | -------------------- | -------------- |
| `~`        | 逐位非（取反） | `~a`                                        | 对每一位进行取反操作（0→1，1→0）       | `~0101 → 1010`       | `~5 = -6`      |
| `&`        | 逐位与         | `a & b`                                     | 同 1 则 1，否则 0                      | `0101 & 0011 → 0001` | `5 & 3 = 1`    |
| \|         | 逐位或         | `a | b`                                     | 有 1 则 1，否则 0                      | `0101 |0011 → 0111`  | `5 | 3 = 7`    |
| `^`        | 逐位异或       | `a ^ b`                                     | 不同则 1，相同则 0                     | `0101 ^ 0011 → 0110` | `5 ^ 3 = 6`    |
| `<<`       | 左移           | `a << shift`（`shift`为偏移量，即移动几位） | 二进制位左移，右边补 0                 | `0101 << 2 → 010100` | `5 << 2 = 20`  |
| `>>`       | 右移           | `a >> shift`                                | 二进制位右移，左边补符号位（正数补 0） | `0101 >> 2 → 0001`   | `5 >> 2 = 1`   |

> [!NOTE]
>
> 位运算都是将原数字转换为二进制补码后进行操作。

## 赋值运算符

赋值运算符用于给变量赋值：

| **运算符** | **功能** | **示例** | **等价表达式** | **运算过程**                                 | **结果**      |
| ---------- | -------- | -------- | -------------- | -------------------------------------------- | ------------- |
| `=`        | 基本赋值 | `x = 10` | -              | 将右侧值赋给左侧变量                         | `x` 变为 `10` |
| `+=`       | 加法赋值 | `x += 5` | `x = x + 5`    | 先将变量与右侧值相加，再赋值给变量           | `x` 变为 `15` |
| `-=`       | 减法赋值 | `x -= 3` | `x = x - 3`    | 先将变量与右侧值相减，再赋值给变量           | `x` 变为 `12` |
| `*=`       | 乘法赋值 | `x *= 2` | `x = x * 2`    | 先将变量与右侧值相乘，再赋值给变量           | `x` 变为 `24` |
| `/=`       | 除法赋值 | `x /= 4` | `x = x / 4`    | 先将变量除以右侧值，再赋值给变量（整数除法） | `x` 变为 `6`  |
| `%=`       | 取模赋值 | `x %= 3` | `x = x % 3`    | 先计算变量与右侧值的余数，再赋值给变量       | `x` 变为 `0`  |

## 自增和自减运算符

| **运算符** | **功能** | **示例** | **等价表达式** | **运算过程**           | **结果（假设初始 x=5）** |
| ---------- | -------- | -------- | -------------- | ---------------------- | ------------------------ |
| `++`       | 前缀递增 | `++x`    | `x = x + 1`    | 先自增 1，再返回变量值 | 返回 `6`，x 变为 `6`     |
| `++`       | 后缀递增 | `x++`    |                | 先返回变量值，再自增 1 | 返回 `5`，x 变为 `6`     |
| `--`       | 前缀递减 | `--x`    | `x = x - 1`    | 先自减 1，再返回变量值 | 返回 `4`，x 变为 `4`     |
| `--`       | 后缀递减 | `x--`    |                | 先返回变量值，再自减 1 | 返回 `5`，x 变为 `4`     |

## 三元运算符

三元运算符（Ternary Operator）是一种简洁的条件表达式，用于根据条件的真假返回不同的值。它是 C++ 中唯一的三目运算符（接受三个操作数），语法格式为：

```cpp
condition ? expression1 : expression2
```

- **condition**：一个布尔表达式，计算结果为 `true` 或 `false`
- **expression1**：如果 `condition` 为 `true`，返回该表达式的值
- **expression2**：如果 `condition` 为 `false`，返回该表达式的值

| **运算符** | **功能** | **示例**                                     | **等价代码**                          | **结果（假设 x=5）**   |
| ---------- | -------- | -------------------------------------------- | ------------------------------------- | ---------------------- |
| `? :`      | 条件求值 | `int max = (x > 10) ? x : 10;`               | `if (x > 10) max = x; else max = 10;` | `max` 被赋值为 `10`    |    
| `? :`      | 赋值简写 | `x = (x % 2 == 0) ? x : x + 1;`              | `if (x % 2 != 0) x = x + 1;`          | `x` 变为 `6`（偶数化） |

## 运算符优先级

运算符有不同的优先级，这决定了表达式的求值顺序：

|             运算符              |       描述       |                   例子                    | 可重载性 |
| :-----------------------------: | :--------------: | :---------------------------------------: | :------: |
|          **第二级别**           |                  |                                           |          |
|              `++`               |   后自增运算符   | `for (int i = 0; i < 10; i++) cout << i;` |  可重载  |
|              `--`               |   后自减运算符   | `for (int i = 10; i > 0; i--) cout << i;` |  可重载  |
|  **第三级别** （从右向左结合）  |                  |                                           |          |
|              `++`               |   前自增运算符   |   `for (i = 0; i < 10; ++i) cout << i;`   |  可重载  |
|              `--`               |   前自减运算符   |   `for (i = 10; i > 0; --i) cout << i;`   |  可重载  |
|               `+`               |       正号       |               `int i = +1;`               |  可重载  |
|               `-`               |       负号       |               `int i = -1;`               |  可重载  |
|               `!`               |     逻辑取反     |              `if (!done) …`               |  可重载  |
|               `~`               |     按位取反     |             `flags = ~flags;`             |  可重载  |
|          **第五级别**           |                  |                                           |          |
|               `*`               |       乘法       |             `int i = 2 * 4;`              |  可重载  |
|               `/`               |       除法       |          `float f = 10.0 / 3.0;`          |  可重载  |
|               `%`               | 取余数（模运算） |            `int rem = 4 % 3;`             |  可重载  |
|          **第六级别**           |                  |                                           |          |
|               `+`               |       加法       |             `int i = 2 + 3;`              |  可重载  |
|               `-`               |       减法       |             `int i = 5 - 1;`              |  可重载  |
|          **第七级别**           |                  |                                           |          |
|              `<<`               |      位左移      |          `int flags = 33 << 1;`           |  可重载  |
|              `>>`               |      位右移      |          `int flags = 33 >> 1;`           |  可重载  |
|          **第九级别**           |                  |                                           |          |
|               `<`               |       小于       |             `if (i < 42) ...`             |  可重载  |
|              `<=`               |     小于等于     |            `if (i <= 42) ...`             |  可重载  |
|               `>`               |       大于       |             `if (i > 42) ...`             |  可重载  |
|              `>=`               |     大于等于     |            `if (i >= 42) ...`             |  可重载  |
|          **第十级别**           |                  |                                           |          |
|              `==`               |       等于       |            `if (i == 42) ...`             |  可重载  |
|              `!=`               |      不等于      |            `if (i != 42) ...`             |  可重载  |
|         **第十一级别**          |                  |                                           |          |
|               `&`               |     位与运算     |           `flags = flags & 42;`           |  可重载  |
|         **第十二级别**          |                  |                                           |          |
|               `^`               |    位异或运算    |           `flags = flags ^ 42;`           |  可重载  |
|         **第十三级别**          |                  |                                           |          |
|               `|`               |     位或运算     |           `flags = flags | 42;`           |  可重载  |
|         **第十四级别**          |                  |                                           |          |
|              `&&`               |    逻辑与运算    |    `if (conditionA && conditionB) ...`    |  可重载  |
|         **第十五级别**          |                  |                                           |          |
|              `||`               |    逻辑或运算    |    `if (conditionA || conditionB) ...`    |  可重载  |
| **第十六级别** （从右向左结合） |                  |                                           |          |
|              `? :`              |    条件运算符    |         `int i = a > b ? a : b;`          | 不可重载 |
|               `=`               |       赋值       |               `int a = b;`                |  可重载  |
|              `+=`               |    加赋值运算    |                 `a += 3;`                 |  可重载  |
|              `-=`               |    减赋值运算    |                 `b -= 4;`                 |  可重载  |
|              `*=`               |    乘赋值运算    |                 `a *= 5;`                 |  可重载  |
|              `/=`               |    除赋值运算    |                 `a /= 2;`                 |  可重载  |
|              `%=`               |    模赋值运算    |                 `a %= 3;`                 |  可重载  |
|              `<<=`              |  位左移赋值运算  |              `flags <<= 2;`               |  可重载  |
|              `>>=`              |  位右移赋值运算  |              `flags >>= 2;`               |  可重载  |
|              `&=`               |   位与赋值运算   |           `flags &= new_flags;`           |  可重载  |
|              `^=`               |  位异或赋值运算  |           `flags ^= new_flags;`           |  可重载  |
|              `|=`               |   位或赋值运算   |           `flags |= new_flags;`           |  可重载  |

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **除零错误**：
```cpp
int main() {
    int x = 10;
    int y = 0;
    int result = x / y;  // 运行时错误
    std::cout << result << std::endl;
    return 0;
}
```

2. **整数溢出**：
```cpp
int main() {
    int max_int = INT_MAX;
    int overflow = max_int + 1;
    std::cout << "Overflow: " << overflow << std::endl;
    return 0;
}
```

3. **浮点数精度问题**：
```cpp
int main() {
    double a = 0.1;
    double b = 0.2;
    double c = a + b;
    std::cout << "0.1 + 0.2 = " << c << std::endl;
    std::cout << "c == 0.3: " << (c == 0.3) << std::endl;
    return 0;
}
```
