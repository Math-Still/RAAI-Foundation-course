# 1.5 输入输出

> 原文：[C++ Input/Output](https://en.cppreference.com/w/cpp/io)

> 译者：RAAI学习团队

C++的输入输出系统基于流的概念。流是数据的序列，可以从源（输入流）读取或写入到目标（输出流）。C++提供了强大的I/O库来处理各种类型的输入输出操作。

## 基本输出

使用`std::cout`进行基本输出：

```cpp
#include <iostream>

int main() {
    // 基本输出
    std::cout << "Hello, World!" << std::endl;
    
    // 输出变量
    int age = 25;
    std::cout << "Age: " << age << std::endl;
    
    // 输出多个值
    std::cout << "Name: " << "John" << ", Age: " << age << std::endl;
    
    // 不使用endl（不换行）
    std::cout << "This is on the same line ";
    std::cout << "as this." << std::endl;
    
    return 0;
}
```

## 基本输入

使用`std::cin`进行基本输入：

```cpp
int main() {
    int age;
    std::string name;
    
    // 输入整数
    std::cout << "Enter your age: ";
    std::cin >> age;
    
    // 输入字符串
    std::cout << "Enter your name: ";
    std::cin >> name;  // 只能读取一个单词
    
    std::cout << "Hello, " << name << "! You are " << age << " years old." << std::endl;
    
    return 0;
}
```

## 格式化输出

使用`std::cout`的格式化功能：

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159265359;
    
    // 设置精度
    std::cout << "Default precision: " << pi << std::endl;
    std::cout << "Fixed precision (2): " << std::fixed << std::setprecision(2) << pi << std::endl;
    std::cout << "Scientific notation: " << std::scientific << pi << std::endl;
    
    // 设置字段宽度和填充
    int number = 42;
    std::cout << "Default: " << number << std::endl;
    std::cout << "Width 10: " << std::setw(10) << number << std::endl;
    std::cout << "Width 10, fill with 0: " << std::setw(10) << std::setfill('0') << number << std::endl;
    
    // 十六进制和八进制输出
    std::cout << "Decimal: " << number << std::endl;
    std::cout << "Hexadecimal: " << std::hex << number << std::endl;
    std::cout << "Octal: " << std::oct << number << std::endl;
    std::cout << "Back to decimal: " << std::dec << number << std::endl;
    
    return 0;
}
```

## 字符串输入

处理包含空格的字符串输入：

```cpp
#include <iostream>
#include <string>

int main() {
    std::string full_name;
    
    // 使用getline读取整行
    std::cout << "Enter your full name: ";
    std::getline(std::cin, full_name);
    
    std::cout << "Hello, " << full_name << "!" << std::endl;
    
    // 混合输入（注意缓冲区问题）
    int age;
    std::string city;
    
    std::cout << "Enter your age: ";
    std::cin >> age;
    
    // 清除输入缓冲区
    std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
    
    std::cout << "Enter your city: ";
    std::getline(std::cin, city);
    
    std::cout << "Age: " << age << ", City: " << city << std::endl;
    
    return 0;
}
```

## 文件输入输出

使用文件流进行文件操作：

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    // 写入文件
    std::ofstream outFile("output.txt");
    if (outFile.is_open()) {
        outFile << "Hello, File!" << std::endl;
        outFile << "This is line 2." << std::endl;
        outFile.close();
        std::cout << "File written successfully." << std::endl;
    } else {
        std::cout << "Error opening file for writing." << std::endl;
    }
    
    // 读取文件
    std::ifstream inFile("output.txt");
    if (inFile.is_open()) {
        std::string line;
        std::cout << "File contents:" << std::endl;
        while (std::getline(inFile, line)) {
            std::cout << line << std::endl;
        }
        inFile.close();
    } else {
        std::cout << "Error opening file for reading." << std::endl;
    }
    
    return 0;
}
```

## 错误处理

处理输入输出错误：

```cpp
int main() {
    int number;
    
    std::cout << "Enter a number: ";
    
    // 检查输入是否成功
    if (std::cin >> number) {
        std::cout << "You entered: " << number << std::endl;
    } else {
        std::cout << "Invalid input!" << std::endl;
        // 清除错误状态
        std::cin.clear();
        // 清除输入缓冲区
        std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
    }
    
    return 0;
}
```

## 流操作符

C++提供了多种流操作符：

```cpp
int main() {
    // endl: 换行并刷新缓冲区
    std::cout << "Line 1" << std::endl;
    
    // flush: 刷新缓冲区
    std::cout << "Line 2" << std::flush;
    
    // ends: 添加空字符
    std::cout << "Line 3" << std::ends;
    
    // ws: 跳过空白字符
    std::cin >> std::ws;
    
    return 0;
}
```

## 如何使它崩溃

尝试以下操作来观察程序的行为：

1. **输入类型不匹配**：
```cpp
int main() {
    int number;
    std::cout << "Enter a number: ";
    std::cin >> number;  // 输入字母而不是数字
    std::cout << "Number: " << number << std::endl;
    return 0;
}
```

2. **文件不存在**：
```cpp
int main() {
    std::ifstream file("nonexistent.txt");
    std::string line;
    std::getline(file, line);  // 尝试读取不存在的文件
    std::cout << line << std::endl;
    return 0;
}
```

3. **缓冲区溢出**：
```cpp
int main() {
    char buffer[10];
    std::cout << "Enter a long string: ";
    std::cin >> buffer;  // 输入超过9个字符的字符串
    std::cout << "Buffer: " << buffer << std::endl;
    return 0;
}
```
